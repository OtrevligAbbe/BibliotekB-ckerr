# Bibliotekshanterare

Detta är en enkel C#-applikation för att hantera ett bibliotek med böcker. Programmet tillåter dig att:

- Lägga till bok:** Ange titel, författare och år för att lägga till en bok.
- Ta bort bok:** Ta bort en bok från biblioteket genom att ange dess titel.
- Lista böcker:** Visa alla böcker som finns i biblioteket.
- Söka efter bok:** Sök efter en bok baserat på titel och visa detaljer om den.
- Sortera böcker:** Sortera alla böcker i biblioteket i alfabetisk ordning baserat på titeln.

## Komma igång

1. **Kör Programmet:** Starta applikationen för att visa huvudmenyn.
2. **Välj Alternativ:** Ange ett nummer för att välja en funktion:
   -1: Lägg till en bok
   -2: Ta bort en bok
   -3 Lista alla böcker
   -4: Sök efter en bok
   -5: Sortera böcker
   -6: Avsluta programmet
3. Följ Instruktionerna: Följ anvisningarna för att utföra åtgärder.

## Exempel på Användning

1. För att lägga till en bok, välj alternativet "1" och ange bokens titel, författare och år.
2. För att lista alla böcker, välj alternativet "3".
3. För att söka efter en bok, välj alternativet "4" och ange bokens titel.
4. För att sortera böckerna i alfabetisk ordning, välj alternativet "5".
